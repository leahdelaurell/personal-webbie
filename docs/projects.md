# Projects

- **Micro‑frontend platform (CoBank)** – architected and delivered a greenfield micro‑frontend ecosystem using RsBuild/Rspack, enabling key customer‑facing workflows to migrate off of legacy SharePoint and into a cloud‑native architecture.

- **Experimentation platform (Charter)** – led the design and delivery of an experimentation system that boosted performance by 108%, standardized A/A/A/B testing across products, and enabled ML model experimentation.
- **DevOps & CI/CD automation** – built GitHub Actions pipelines, Kubernetes deployments, and infrastructure as code patterns that support rapid delivery and high reliability.
